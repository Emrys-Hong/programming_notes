## exception Handling
```java
try {
   // Protected code
} catch (ExceptionType1 e1) {
   // Catch block, the smallest exception,
} catch (ExceptionType2 e2) {
   // Catch block, medium exception
} catch (ExceptionType3 e3) {
   // Catch block, big exception.
}
```
